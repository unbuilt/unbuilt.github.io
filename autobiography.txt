
个人小传

# 1979年秋出生于河北省一个偏远的小山村，从小在那里长大，淳朴的民风赋予了我勤奋朴实的性格。
# 小学六年从山村到矿区到县城，换了三所学校就读，锻炼了我的适应能力。
# 1992年升入初中，三年的踏实努力为以后的学习打下了良好的基础。
# 1995年考入郑州测绘学校，四年的学习与实践使我成为一名测绘工作者。第一次见到了电脑，学习了写程序。
# 1999年从黄河南岸来到珠江边，短暂的停留让我看到了更广阔的世界。第一次用鼠标，写了第一个GUI Hello World程序。
# 2000年徘徊于北京的大街小巷，感受着科技的气息。每天用AutoCAD绘图，写VB脚本处理数据。
# 2001年分配到县国土管理局工作，每天挨家挨户进行土地登记发放土地证。跟形形色色的人交流。
# 2001年秋放弃工作，来到合肥继续学习，在中国科学技术大学一晃七年过去了。
# 2008年毕业，加入科大讯飞从事软件开发。
# 2010年加入创业公司，了解怎样让计算机更聪明。
# 2012年冬进入微软，开始虚拟个人助理的开发。
